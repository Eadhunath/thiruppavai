# Thiruppavai

A simple website to read the verses of Thiruppavai during the Margazhi month.

## Features

- **Three Languages** - Read in Tamil, Sanskrit, or English transliteration
- **Daily Verse** - Automatically shows the verse for today's date during Margazhi (December-January)
